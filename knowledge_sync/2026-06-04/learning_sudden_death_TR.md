## 核心机理

### 热失控根本路径
外部滥用(机械/电/热) + 内部缺陷 → 局部内短路 → 温度累积 → 热失控

### 内短路风险排序
铝箔参与的内短路 > 其他模式（风险最高）

### 猝死型热失控定义
- 电压/温度正常 → 约3分钟内电压异常 → 整车级温度报警
- 发生前容量/内阻参数均符合生产标准，无渐变离群征兆
- 分两阶段：无数据表征的内部孕育期 + 数秒级的失效爆发期

### 孕育期机理
制造缺陷/多因素工况叠加 → 局部锂析出 → 锂枝晶成百上千次循环缓慢生长 → 刺穿/压破隔膜 → 电压突降+局部急剧温升 → SEI膜分解+正极释氧 → 热失控

## 多维防控体系

### 设计端
- N/P比、涂布面密度、压实密度
- 配方稳定性、隔膜选型
- 对齐度、盖板设计、绝缘防护
- 提升电池对异物的容忍度（异物鲁棒性）

### 制造端（三大环节）
- 电极制备：金属异物、电极漏金属
- 电芯装配：焊接质量、极耳控制、入壳干涉
- 化成：负压控制、密封钉焊接、注液量、添加剂管控

### 使用端
- 早期间歇性大电流
- BMS控制不当→微过充/过放
- 低/高SOC长期搁置

## 检测与预警

### 制造检测
- 二维/三维CT：极片翻折、碎裂、对齐度等缺陷精准检出

### 使用端预警（大数据）
- 冲高放低现象：低SOC时电压偏低、高SOC时电压偏高
- 缓慢自放电特性
- 单体电压在整包中排序变化（拉长时间维度分析）
- 三类风险电芯：前两类可通过排序+自放电识别，第三类（完全无表征）仍在探究

## 系统化防控策略
1. 设计先行：材料本征安全 → 结构防制造缺陷
2. 制造保障：全工序风险识别 → 安全边界前置到研发设计 → 零流出
3. 精准检测：CT全覆盖
4. 异物管控：成立专门组织 → 异物标准与地图 → 研发设计阶段前置 → 密闭设备 → 反向溯源
